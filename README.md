## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Fitness Provider OAuth

Set these server-only environment variables before using the Strava or WHOOP connection buttons:

```bash
STRAVA_CLIENT_ID=
STRAVA_CLIENT_SECRET=
WHOOP_CLIENT_ID=
WHOOP_CLIENT_SECRET=
```

Register these callback URLs in the provider dashboards:

```txt
http://localhost:3000/api/oauth/strava/callback
http://localhost:3000/api/oauth/whoop/callback
https://YOUR_PRODUCTION_DOMAIN/api/oauth/strava/callback
https://YOUR_PRODUCTION_DOMAIN/api/oauth/whoop/callback
```

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.
